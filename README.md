# Curse Fox
A Discord bot that adds a little something to those misbehaving or interested users by adding 'curses' to their messages!

## Commands
### Help
```m!Help```

### Cure
```m!Cure <user>```
```m!MassCure```

### Curse
```m!Curse <user> <curse>```

### List Information
```m!Curses```
```m!Cursed```
```m!Cursed All```
```m!Cursed <user>```

### Creating and Deleting curses
```m!Create <curse>```
```m!Delete <curse>```
